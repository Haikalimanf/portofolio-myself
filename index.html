Sertifikat Keahlian

Berikut berbagai sertifikat yang telah saya peroleh sebagai bukti dedikasi dan pencapaian dalam berbagai bidang.

Belajar Membuat Aplikasi Android Pemula

Materi yang dipelajari dalam kelas ini meliputi pengenalan Android Studio, Activity, Intent, View dan ViewGroup, Style dan Theme, serta RecyclerView. Di akhir kelas, siswa diharapkan dapat membuat aplikasi Android yang dapat menampilkan list dan detail data.

Memulai Pemrograman Dengan Kotlin

Memperkenalkan sejarah Kotlin, mengapa harus mempelajari Kotlin, karakteristik Kotlin, dan bagaimana Kotlin berperan dalam pengembangan aplikasi. Akan dibahas juga tentang ekosistem Kotlin

Belajar Dasar Manajemen Proyek

Menafsirkan definisi manajemen proyek secara luas dan menjelaskan penerapan manajemen proyek pada kegiatan sehari-hari. Pada materi ini Anda akan mempelajari peran manajer proyek, latihan membuat resume, memahami nilai seorang manajer proyek, tanggung jawab manajer proyek, kerjasama lintas fungsi, dan kompetensi utama yang dibutuhkan oleh seorang manajer proyek.

Memulai Dasar Pemrograman Untuk Menjadi Pengembang Software

Kelas ini mengajarkan materi dasar pemrograman, seperti sintaksis bahasa pemrograman, variabel, tipe data, logika komputer, dan JavaScript versi ES6. Selain itu, kelas ini juga mengajarkan materi tentang modifikasi aplikasi perangkat lunak, seperti bahasa markah HTML versi HTML5 dan bahasa pemrograman CSS versi 3.

Belajar Pemrograman Java dari Pemula sampai Mahir

Course ini akan membahas serangkaian topik yang komprehensif dalam pemrograman Java. Mulai dari Java Dasar, yang mencakup tipe data, percabangan, perulangan, hingga pembuatan fungsi dan parameter. Selanjutnya, pembahasan berkembang ke Java Object Oriented Programming dengan fokus pada pembuatan class, objek, interface, pewarisan, polimorfisme, dan penanganan error.

Belajar GIT Dasar dalam Pemrograman

Course ini berkenalan dengan Git, mengetahui manfaat penggunaan Git dalam mengelola file, proyek dan juga manfaatnya ketika berkolaborasi dengan tim. Selain itu pada course ini juga dibahas tahapan instalasi Git, konfigurasi dasar, membuat repository, menyalin repository, mengambil dan mengirim perubahan dari proyek.
